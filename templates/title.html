Book Recomendation system using NLP

In this exercise , I have attempted to demo a simple content based recomender system I implemented as follows -

  • Extracted the list of book titles , authors , isbn # from here : Kaggle
  • For each isbn from the dataset , extracted the text snippet by making a rest call to GoogleReads API( in python). Sample Call - https://www.googleapis.com/books/v1/volumes?q=isbn:
  • Removed all the books were the text snippet was empty and finished preparing the dataset and exported it to a pickle file
  • Initialized CountVectorizer using the soup of words as - authors, title and text snippet for each book. Then implemented a method which takes the input as the choice of 1 book selected by user and then calculates the cosine similarity score of all other books from the dataset and returns top 5
